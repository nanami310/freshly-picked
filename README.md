# もぎたて

## 環境構築
1. git clone git@github.com:coachtech-material/laravel-docker-template.git
2. DockerDesktopアプリを立ち上げる
3. docker-compose up -d --build
4. docker-compose exec php bash
5. composer install
6. cp .env.example .env
7. envに以下の環境変数を追加

## 使用技術（実行環境）
- PHP 8.0
- Laravel 10.0
- MySQL 8.0
